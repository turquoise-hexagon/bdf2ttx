-------
bdf2ttx
-------

convert bdf to ttx
==================

:date: March 2021
:version: 0.0
:manual section: 1
:manual group: General Commands Manual

synopsis
--------
bdf2ttx `<file>`

description
-----------
convert a file in bdf format to ttx format

example
-------
::

    $ bdf2ttx banana.bdf > banana.ttx
