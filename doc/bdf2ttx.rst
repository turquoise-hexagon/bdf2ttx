-------
bdf2ttx
-------

convert BDF files to TTX
========================

:date: March 2021
:version: 0.0
:manual section: 1
:manual group: General Commands Manual

synopsis
--------
bdf2ttx `<file>`

description
-----------
bdf2ttx converts BDF (Bitmap Distribution Format) files to TTX (TrueType XML)

example
-------
::

    $ bdf2ttx banana.bdf > banana.ttx
